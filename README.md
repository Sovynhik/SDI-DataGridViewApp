# Лабораторная работа №4 — DataGridViewLab
**Дисциплина:** Проектирование программного интерфейса  
**Вариант 16**

Простое Windows Forms-приложение на C#, реализующее функционал **Телефонной книги**. В работе используется компонент `DataGridView` для отображения и управления контактами, а также реализован механизм сохранения и загрузки данных в/из **XML-файла** с использованием объектов `DataSet` и `DataTable`.

### Что реализовано
- **CRUD-операции** с контактами: добавление, редактирование, удаление и полная очистка таблицы.
- **Персистентное хранение данных**: сохранение и загрузка контактов в локальный **XML-файл** (`phonebook.xml`).
- **Автозагрузка**: проверка наличия и загрузка данных из `phonebook.xml` при старте приложения.
- **Поиск**: функция поиска контактов по полю **Организация** с отображением результатов в отдельной модальной форме (`ResultForm`).
- **Синхронизация полей**: при выборе строки в `DataGridView` данные контакта автоматически заполняют поля ввода.
- **Интерфейс**: Все окна по центру экрана, фиксированный размер, без кнопки «Развернуть».
- **Чистый код, осмысленные имена, комментарии** на русском языке.

### Структура проекта
```

```
